Run the following scripts in this order:

	1. Runs scProportionTest on the tissue atlas to generate differences in cell type proportions between sexes.
			- 1_Cell_Type_Proportion_Analysis.R
	
	2. Generates cell type proportion pie charts per tissue and sex. Plots median number of unique genes expressed per cell type and sex.
		    - 2_Cell_Type_Pie_Chart_Median_Gene_Exp.R
